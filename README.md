# Roteamento-React-Router
Como Executar o Projeto
Para rodar a aplicação em sua máquina local:

1. Pré-requisitos
Ter o Node.js instalado no computador.

2. Clonar o Repositório
Usando o Git digite o comando:

git clone https://github.com/andersonxthgg/Roteamento-React-Router.git

3. Instalar as Dependências
Abra o terminal do windows, dentro da pasta do projeto "Roteamento-React-Router" e execute esse comando: 

npm install

Ele irá baixar a pasta node_modules.

4. Iniciar a Aplicação
Após a instalação, inicie o servidor de desenvolvimento com esse comando:

npm start

O site web irá abrir automaticamente no seu navegador no endereço http://localhost:3000.
